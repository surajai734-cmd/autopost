Complete API Keys Guide

1 Telegram Bot Token

1. Telegram में @BotFather search करें
2. /newbot command send करें
3. Bot name दें (AutoSocialSync)
4. Bot username दें (जैसे: my_autopost_bot)
5. Token copy करें (123456:ABC-DEF...)

2 Facebook & Instagram Keys

1. https://developers.facebook.com पर जाएं
2. Create App → Business type select करें
3. App Name: AutoSocialSync
4. App ID और App Secret note करें
5. Graph API Explorer में जाकर Page Access Token generate करें
6. Instagram के लिए Facebook Business Suite में Instagram account connect करें

3 Twitter API Keys

1. https://developer.twitter.com पर जाएं
2. Apply for Developer Account (Hobbyist)
3. Project बनाएं: AutoSocialSync
4. App create करें
5. Read & Write permissions select करें
6. 4 keys generate करें और save करें

4 Pinterest API Keys

1. https://developers.pinterest.com पर जाएं
2. Create App → AutoSocialSync
3. OAuth tokens generate करें
4. Board ID प्राप्त करने के लिए Pinterest profile पर जाएं
5. Board open करें और URL से ID copy करें
